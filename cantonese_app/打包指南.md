# 📱 粤语学习助手 - 快速打包指南

## Android 快速打包（Windows/Mac/Linux）

### 1. 准备工作

确保已安装：
- ✅ Flutter SDK
- ✅ Android Studio 和 Android SDK
- ✅ Java JDK 11+

检查环境：
```bash
cd cantonese_app
flutter doctor
flutter pub get
```

### 2. 一键打包 APK

```bash
# 进入项目目录
cd cantonese_app

# 清理并构建
flutter clean
flutter pub get
flutter build apk --release
```

### 3. 找到 APK 文件

打包完成后，APK 文件在这里：
```
cantonese_app/build/app/outputs/flutter-apk/app-release.apk
```

### 4. 安装到手机

**方法一：USB 连接安装**
```bash
# 连接手机，启用 USB 调试，然后运行：
flutter install
```

**方法二：直接传输安装**
1. 将 `app-release.apk` 文件复制到手机
2. 在手机上打开文件管理器
3. 点击 APK 文件进行安装
4. 如果提示"未知来源"，需要在设置中允许安装

**方法三：使用 ADB**
```bash
adb install build/app/outputs/flutter-apk/app-release.apk
```

## iOS 快速打包（仅限 Mac）

### 1. 准备工作

确保已安装：
- ✅ macOS 系统
- ✅ Xcode（从 App Store 安装）
- ✅ CocoaPods：`sudo gem install cocoapods`

### 2. 配置项目

```bash
cd cantonese_app
flutter pub get
cd ios
pod install
cd ..
```

### 3. 打开 Xcode 配置

```bash
# 打开项目
open ios/Runner.xcworkspace
```

在 Xcode 中：
1. 选择 Runner 项目
2. 在 "Signing & Capabilities" 中：
   - 选择您的 Apple Developer Team
   - 设置唯一的 Bundle Identifier
   - 勾选 "Automatically manage signing"

### 4. 构建并打包

**方法一：命令行**
```bash
flutter build ios --release
# 然后通过 Xcode Archive 导出 IPA
```

**方法二：Xcode GUI**
1. 在 Xcode 中选择 Product > Archive
2. 等待构建完成
3. 在 Organizer 中选择 Archive
4. 点击 "Distribute App"
5. 选择分发方式（Ad Hoc 用于测试，App Store 用于发布）

## ⚠️ 重要提示

### ✅ 应用已完全独立运行！

**好消息！** 应用现在**不需要本地后端服务器**了！

应用直接调用 AI API（如 SiliconFlow、OpenAI 等），用户只需在设置中配置：
- API Key
- Base URL（如 `https://api.siliconflow.cn/v1`）
- 模型名称

### 用户配置步骤

1. 安装应用后，打开设置
2. 输入 API Key 和 Base URL
3. 点击"获取模型列表"
4. 选择模型并保存
5. 开始使用！

### 支持的 API 服务

- **SiliconFlow**（推荐）：`https://api.siliconflow.cn/v1`
- **OpenAI**：`https://api.openai.com/v1`
- 任何 OpenAI 兼容的 API 服务

## 🎯 常见问题

**Q: 构建失败怎么办？**
```bash
flutter clean
flutter pub get
flutter doctor
```

**Q: Android 安装失败？**
- 检查手机是否允许"未知来源"安装
- 卸载旧版本再安装
- 检查存储空间是否足够

**Q: iOS 需要开发者账号吗？**
- 免费账号：只能安装到自己的设备，7 天有效期
- 付费账号（$99/年）：可以发布到 App Store

**Q: 应用无法连接后端？**
- 检查 `api_service.dart` 中的 `baseUrl`
- 确保后端服务正在运行
- 检查防火墙设置

## 📦 发布到应用商店

### Google Play
1. 创建 Google Play Console 账号
2. 上传 `app-release.aab`（运行 `flutter build appbundle`）
3. 填写应用信息并提交审核

### App Store
1. 创建 App Store Connect 账号
2. 通过 Xcode 上传 IPA
3. 填写应用信息并提交审核
